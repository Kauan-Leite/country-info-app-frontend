This is the FrontEnd to Country-Info-APP

## :dna: To Exec


### 1 - Clone the repo:

```
git clone git@github.com:Kauan-Leite/country-info-app-frontend.git
```

### 2 - No terminal do repositório, rode o comando:

    docker-compose up -d

### 3 - Agora basta acessar a aplicação em:

    http://localhost:3000/
